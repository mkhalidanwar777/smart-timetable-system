# Smart Timetable System

An AI-powered university timetable management system developed using Flask and MySQL. The system helps automate timetable generation, teacher scheduling, course allocation, and conflict management for educational institutions.

## Features

- Teacher and HOD login system
- Course management
- Teacher availability scheduling
- Conflict-free timetable generation
- Classroom and lab allocation
- MySQL database integration
- Responsive web interface

## Technologies Used

- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript

## Project Structure

- `routes/` → Application routes
- `models/` → Database models
- `templates/` → HTML templates
- `static/` → CSS, JS, and assets
- `database/` → SQL schema and seed files

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python app.py
```
